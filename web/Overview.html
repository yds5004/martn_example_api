Blog Search API

Parameter

qry
User query
si
The ID of the first document which is printed in a page
li
The ID of the last document which is printed in a page
n
The document count which is printed in a page

Example

	            http://tyche.asia/search.map?cmd=search&qry=티스토리&si=0&li=9&n=10

Weekly Tag Set

Parameter

nweek
Weeks to get tags
size
Tag Count

Example

            	http://tyche.asia/search.map?cmd=search&nweek=1&size=10
